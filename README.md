# Bedrock Phone

**Native AWS Bedrock Client for iOS**

Bedrock Phone is a powerful, privacy-focused iOS client that connects directly to your AWS Bedrock account. Chat with the latest AI models without intermediary servers or monthly subscriptions.

## Features

*   **Private & Secure**: Connects directly to AWS. Your API keys are stored securely in the iOS Keychain and never leave your device.
*   **Bring Your Own Keys (BYOK)**: Pay wholesale prices directly to Amazon. No markup, no subscription.
*   **Voice Chat**: Speak naturally to the AI with on-device speech recognition.
*   **File Analysis**: Upload and analyze PDFs, text files, and source code (`.swift`, `.py`, `.js`, etc.).
*   **Code Highlighting**: Beautiful syntax highlighting for code snippets generated by the AI.

## Supported Models

Bedrock Phone allows you to switch between the most advanced models available on AWS Bedrock.

**Currently Supported:**
*   **Claude 4.5 Family** (Opus, Sonnet, Haiku)
*   **Claude 4 Family** (Sonnet)

*Note: Legacy Claude 3/3.5 models are not currently available in this version. Support for additional models is coming in future releases.*
